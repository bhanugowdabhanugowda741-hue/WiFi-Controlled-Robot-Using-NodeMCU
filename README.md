# WiFi Controlled Robot Using NodeMCU

## Project Overview

WiFi Controlled Robot Using NodeMCU is an IoT based robotic system designed for wireless remote navigation using internet connectivity. The robot can be controlled through a web interface or mobile device over WiFi, enabling long-distance directional movement without physical contact.

This project integrates NodeMCU microcontroller, ESP8266 WiFi communication, L293D motor driver, and DC motors to achieve real-time robotic control. The system demonstrates practical applications in surveillance, defense support, hazardous environment monitoring, and remote automation.

---

## Objective

The main objective of this project is to build a cost-effective wireless robotic vehicle that can be remotely controlled through internet communication beyond the limited range of Bluetooth or RF modules.

## Working Principle

The NodeMCU microcontroller connects to WiFi and receives movement instructions from the user through a wireless interface. Based on the command received, the L293D motor driver rotates the connected DC motors in required directions.

Robot directional controls include:

* Forward
* Backward
* Left
* Right
* Stop
